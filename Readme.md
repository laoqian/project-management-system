<a name="系统简介"></a>
## 📋 系统简介

> **集成化项目管理系统 3.0** 是一套专为**工程企业、EPC 总承包商、境外施工单位**设计的全流程业务管理系统，深度支持国内外工程项目的**一体化数字化管理**。

系统以**合同结算、材料核销、费用控制**为核心，覆盖工程项目从合同签订到竣工结算的全生命周期；同时提供**多语言、多币种、多公司**的国际化能力，满足跨国工程项目的复杂管理需求。

### 📊 数据实时联动，告别传统线下报表

> 传统 Excel 报表存在**数据分散、版本混乱、无法实时更新**的问题。系统将所有业务数据统一管理，实现**数据一次录入、多方联动、实时统计**。

|  | 传统线下 Excel 报表 | 本系统 |
|:--|-------------------|--------|
| **数据来源** | 各人各自维护，版本多、易出错 | 统一数据库，数据唯一、实时同步 |
| **数据更新** | 手动更新，滞后严重 | 业务操作即更新，实时反映最新状态 |
| **统计口径** | 各部门口径不一致，汇总困难 | 统一统计口径，系统自动汇总 |
| **查询效率** | 翻找多个文件，耗时费力 | 多条件组合查询，秒级响应 |
| **权限控制** | 文件共享，难以精细控制 | 按角色、项目、公司精细授权 |
| **审批留痕** | 纸质签字或邮件，难以追溯 | 系统全程记录，可按流程、单据追溯 |
| **跨部门协作** | 邮件或 U 盘传递，效率低 | 系统内数据共享，跨部门实时协作 |
| **报表输出** | 手动制作，格式不统一 | 内置报表引擎，一键生成标准报表 |

**数据联动示例**

```
合同签订 ──→ 合同金额自动计入实时概算（无需手动更新）
采购结果 ──→ 自动关联合同报批，数据自动带入（无需重复录入）
材料入库 ──→ 库存数据实时更新，库存台账自动生成（无需手工登记）
费用报销 ──→ 审批通过后自动更新项目成本（实时反映预算执行）
```

### 1️⃣ 合同管理

> 覆盖合同从签订到结算的全生命周期，是系统的**核心业务模块**。

#### 功能总览

| 子模块 | 功能说明 | 业务价值 |
|--------|----------|----------|
| **项目维护** | 登记项目基础信息（工程名称、概算金额、预算金额、工期等） | 项目信息集中管理，便于查询统计 |
| **合同台账** | 多层级台账（项目 → 年度 → 概算分类 → 合同），支持双击查看详情 | 清晰展示项目→合同关系，快速定位 |
| **合同结算** | 结算管理，支持结算编号/计量编号查询，**支持结算打印** | 合同结算全流程管理，支持打印输出 |
| **实时概算** | 跟踪项目概算执行情况（概算/预算/合同金额、累计进度款、累计结算款） | 实时掌握项目成本，避免超预算 |
| **变更管理** | 合同变更事项记录与审批 | 变更全程记录，避免纠纷 |
| **合同列表** | 统一查询入口，支持多条件组合查询 | 快速查询合同信息 |

#### 合同详情页（8 个标签页）

双击合同台账中的合同，可查看合同详情，包含以下 8 个标签页：

```
┌────────────┬────────────┬────────────────┬────────────┐
│ ① 合同主要 │ ② 清单     │ ③ 核销材料表   │ ④ 结算汇总 │
│    信息     │            │  （核心功能）   │            │
├────────────┼────────────┼────────────────┼────────────┤
│ ⑤ 用户授权 │ ⑥ 费用分摊 │ ⑦ 材料核销配合比│ ⑧ 附件     │
│            │            │  （核心配置）   │            │
└────────────┴────────────┴────────────────┴────────────┘
```

#### 合同结算流程

```
合同签订 → 计量签证 → 材料核销 → 结算编制 → 结算审批 → 结算打印 → 合同归档
```

#### 特色功能

- ✅ **合同结算打印**：支持支付汇总、结算报审表等多种单据打印
- ✅ **EPC 合同结算打印**：EPC 合同的独立业主签章布局（HSK 环境专属）
- ✅ **合同触发器**：合同名称/编号变更时自动同步更新关联表
- ✅ **结算专业工程师搜索**：结算列表支持按专业工程师姓名搜索

### 2️⃣ 费用控制

> 工程企业的**核心业务模块**，通过计量、核销、结算、归概等环节实现费用的全过程管控。

#### 功能总览

| 子模块 | 功能说明 | 业务价值 |
|--------|----------|----------|
| **计量(台账)** | 6 个签证类型：工程量/电费/考核/材料/盘库/水费 | 全面管理各类签证，确保工程量准确 |
| **核销(台账)** | 项目核销管理，关联合同与发票<br/>**材料核销是核心功能** | 确保材料成本准确，避免成本漏洞 |
| **结算(台账)** | 项目结算管理，支持结算打印 | 合同结算全流程管理，支持打印输出 |
| **归概(台账)** | 将结算费用归集到概算 | 实时掌握项目成本，避免超预算 |

#### 费用控制流程

```
工程量计量 → 签证确认 → 材料核销 → 结算编制 → 结算审批 → 归概分析
```

#### 计量(台账) — 6 个签证类型

| 签证类型 | 说明 | 业务价值 |
|----------|------|----------|
| 工程量签证 | 工程量签证管理 | 准确记录工程量，为结算提供依据 |
| 电费签证 | 电费签证管理 | 准确记录电费，为结算提供依据 |
| 考核签证 | 考核签证管理 | 准确记录考核结果，为结算提供依据 |
| 材料签证 | 材料签证管理 | 准确记录材料用量，为材料核销提供依据 |
| 盘库签证 | 盘库签证管理 | 准确记录盘库结果，确保账实相符 |
| 水费签证 | 水费签证管理 | 准确记录水费，为结算提供依据 |

#### 材料核销（核心功能）

**核销流程**

```
材料入库 → 材料领用 → 材料消耗 → 材料核销 → 成本归集
```

**配合比管理**
- 配合比配置（用于材料核销的配合比数据）
- 材料核销时，系统根据配合比自动计算核销金额

**业务价值**
- ✅ 确保材料成本准确，避免成本漏洞
- ✅ 自动计算核销金额，提升工作效率
- ✅ 配合比管理，确保核销准确性

### 3️⃣ 设备物资

> 覆盖物资从入库到出库的全流程，特别是**材料核销相关的物资管理**。

| 子模块 | 功能说明 | 业务价值 |
|--------|----------|----------|
| **入库单台账** | 入库单据管理（供应商、物资明细、金额等） | 物资入库信息准确记录，便于追溯 |

| **库存日志** | 每一笔出入库的详细记录 | 库存变动全程记录，便于审计和追溯 |

| **物资信息** | 物资基础信息（编码、名称、规格、单位、单价等） | 物资信息统一管理，避免重复录入 |

| **清关费用核算** | 清关费用核算 | 清关费用准确核算，避免成本漏洞 |

#### 业务价值

- ✅ 物资全流程管理，从入库到出库一目了然
- ✅ 实时库存查询，避免物资短缺或积压

- ✅ 库存日志完整记录，便于审计和追溯

> 采购业务的全流程管理，从采购计划到合同报批的完整闭环。

| 子模块 | 功能说明 | 业务价值 |

#### 工具栏功能

- 导入合同（从 ECP 平台导入合同信息）
- 多条件查询（项目名称、采购方式、审批状态等）

#### 业务价值

- ✅ 采购全流程闭环管理，透明可控
- ✅ 采购计划与预算挂钩，有效控制采购成本
- ✅ 支持打印、作废、导入合同等操作，提升工作效率
- ✅ 采购结果记录完整，便于审计和追溯

### 多语言支持

- 系统界面支持**中文 / 英文**双语切换，用户可随时在个人设置中切换语言
- 所有菜单、按钮、字段标签均已完成双语翻译，切换后全界面同步生效
- 管理员可在「系统设置 → 语言管理」中维护/扩展翻译词条，无需修改代码

### 多币种支持

系统内所有涉及金额的业务模块（合同、采购、费用报销、结算等）均原生支持多币种：

| 币种 | 代码 | 说明 |
|------|------|------|
| 人民币 | CNY | 国内项目默认货币 |
| 美元 | USD | 国际工程常用货币 |
| 印尼盾 | IDR | 东南亚工程专属支持 |
| 其他币种 | — | 可由管理员在「汇率管理」中自行新增 |

- **汇率管理**：按生效日期维护汇率，支持追溯历史汇率，确保历史单据金额不受汇率波动影响
- **多币种结算**：合同、采购、费用报销均可选择对应币种，系统自动记录原币金额，方便对账
